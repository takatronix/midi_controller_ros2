ドキュメントを更新する際は、以下の点に注意してください：

1. **日本語での記述** - 全てのドキュメントは日本語で記述
2. **実用的な例** - 実際に動作するサンプルコードを含める
3. **トラブルシューティング** - よくある問題と解決方法を記載
4. **バージョン管理** - ドキュメントの更新日時とバージョンを記録
